#### webpack5-vue3-mobile

基于 `vue` + `pinia` + `vant` + `sass` 构成

#### 环境变量`.env`

|  环境变量   |          描述          |
| :---------: | :--------------------: |
| OUTPUT_DIR  | 打包目录（默认`dist`） |
| PROJECT_URL |  DNS 预加载 （域名）   |
|  BASE_URL   |  请求地址（开发环境）  |
|  PROD_URL   |  请求地址（正式环境）  |

#### 环境变量`BN_ENV`

|  `dev`   |  `test`  |  `prod`  |
| :------: | :------: | :------: |
| 开发环境 | 测试环境 | 正式环境 |

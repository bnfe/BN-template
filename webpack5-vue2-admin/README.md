#### webpack5-vue2-admin

基于 `vue` + `vuex` + `element-ui` + `sass` 构成

#### 环境变量`.env`

|  环境变量   |           描述           |
| :---------: | :----------------------: |
| OUTPUT_DIR  |  打包目录（默认`dist`）  |
|  BASE_URL   | 请求地址（开发环境可用） |
| PROJECT_URL |   DNS 预加载 （域名）    |

#### 环境变量`BN_ENV`

|  `dev`   |  `test`  |  `prod`  |
| :------: | :------: | :------: |
| 开发环境 | 测试环境 | 正式环境 |
